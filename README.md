# HIV-1 Genome Browser

## Introduction
This project focuses on building an interactive genome browser for Human Immunodeficiency Virus type 1 (HIV-1) to explore genomic, structural, and functional features. HIV-1 is a highly mutable retrovirus with significant global health impact, making it critical to study its genomic variability, drug resistance mutations, and evolutionary patterns.

The goal of this project is to create a unified platform that allows researchers to visualize HIV-1 genomic data, compare subtypes, analyze mutations, and explore structural features to support therapeutic development and vaccine research.

- **Phylogenetic Analysis Tools**
- **3D Protein Visualization Tools**
- **Git / GitHub**
- **AWS**


## Features
- Interactive genome browser for HIV-1  
- Annotation tracks for key genes (gag, pol, env, rev, nef, etc.)  
- Visualization of subtype variation (A, B, C, CRFs)  
- Tracking of drug resistance mutations (DRMs)  
- 3D protein structure visualization (RT, protease, integrase, Env)  
- Multiple sequence alignment (MSA) for mutation analysis  
- Phylogenetic tree visualization for evolutionary relationships  


## Process
1. Selected HIV-1 due to its high mutation rate and global health impact  
2. Collected genomic and protein data across multiple HIV-1 subtypes  
3. Built genome browser using JBrowse2  
4. Integrated annotation tracks for genomic features and genes  
5. Added MSA functionality to analyze sequence variation  
6. Incorporated phylogenetic trees to study evolutionary relationships  
7. Integrated 3D protein structures for functional and structural insights  
8. Enabled comparison across subtypes and identification of mutation hotspots  


## Results
- Developed an interactive platform for exploring HIV-1 genomic and structural data  
- Enabled visualization of subtype diversity and geographic variation  
- Identified key mutation hotspots in genes like **env** and **pol**  
- Provided tools to analyze drug resistance mutations affecting therapies  
- Created a unified system for genomic, structural, and evolutionary analysis  


## What I Learned

- The complexity of viral evolution and mutation dynamics  
- How genetic variation impacts drug resistance and vaccine design  
- The importance of integrating multiple data types (genomic, structural, evolutionary)  
- How visualization tools enable deeper biological insights
